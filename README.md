#  SchoolLife

NEIS Open API를 활용한 **급식 · 시간표 조회 iOS 앱**  
위젯을 사용하여 급식·시간표 조회

---

## 개요

기존 시간표 및 급식 조회 서비스는 접근성이 떨어지고 번거로움.
이를 해소하고자 iOS에서 동작하는 급식·시간표 조회 및 위젯 시스템 개발.

이 프로젝트는 **학교, 학년, 반**을 설정하면
- 급식
- 시간표

를 **앱과 위젯**에서 즉시 확인할 수 있도록 제작된 프로젝트이다.

---

## 기술 스택

- **Language**: Swift
- **UI**: SwiftUI
- **Widget**: WidgetKit
- **API**: NEIS Open API
- **Data Sharing**: App Groups (UserDefaults)
- **IDE**: Xcode

---

## 주요 기능

- 학교명 검색 및 선택
- 학년 · 반 설정
- 날짜별 급식 조회
- 날짜별 시간표 조회
- 홈 화면 위젯 제공
  - 급식 위젯
  - 시간표 위젯
